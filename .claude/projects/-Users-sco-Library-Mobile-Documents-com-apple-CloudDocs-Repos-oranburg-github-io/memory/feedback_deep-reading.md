Read every page of each paper before writing an insight post. A 5-page skim captures the thesis; a full read captures the original contribution. The return is 3-5x insight density.

**Why:** On 2026-03-30, the Exclusive Inclusion post was rewritten after full 25-page reading. The skim missed: the disentanglement matrix (the paper's most powerful tool), the immunodeficiency analogy (the structural argument), option value theory (why universities capitulate on compliance but resist governance reform), the interlocking reform architecture, and all specific evidence (boycott endorser counts, settlement dollar figures, court quotes).

**How to apply:**
1. Read all pages before writing. Not pages 1-5.
2. Look for the original analytical move — what does this paper do that no other paper does?
3. Extract specific evidence: numbers, case names, dollar figures, court quotes.
4. Find the paper's best sentence and use it as the hook or closing.
5. Write a reflection entry in `new-style/deep-read-reflections.md` after each paper.
6. Plan the next paper's reading by listing questions to answer before starting.
